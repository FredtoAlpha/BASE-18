# Tests Directory

This directory contains all test files for the BASE-18 project.

## Test Files

- **TEST_PARITE_ADAPTATIVE.js** - Tests for adaptive parity optimization (Phase 3)
- **TEST_CONSOLIDATION.js** - Tests for consolidation functionality
- **GroupsModule_TestCases.js** - Test cases for the groups module
- **Phase3_PariteAdaptive_Tests.js** - Additional tests for Phase 3 adaptive parity
- **TEST_REGEX_PATTERNS.js** - Tests for regex pattern matching

## Running Tests

These are standalone test files meant to be executed manually in Google Apps Script environment.

## Notes

All test files are manually executed and not part of an automated test suite.
